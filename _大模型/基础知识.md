


RAG（Retrieval-Augmented Generation，检索增强生成） 是一种结合了信息检索技术与语言生成模型的人工智能技术。该技术通过从外部知识库中检索相关信息，并将其作为提示（Prompt）输入给大型语言模型（LLMs），以增强模型处理知识密集型任务的能力，如问答、文本摘要、内容生成等。RAG 模型由 Facebook AI Research（FAIR）团队于 2020 年首次提出，并迅速成为大模型应用中的热门方案。

# MCP

MCP（Model Context Protocol，模型上下文协议）是由 Anthropic 于2024年11月提出的开源协议，旨在为大语言模型（LLM）与外部数据源、工具和服务之间的交互提供标准化接口。


在传统的开发模式中，不同的大模型对应不同的调用方法（即模型推理与工具调用是绑定的），在 MCP 开发模式下，MCP 通过协议层将工具调用与模型推理分离，MCP 把不同的大模型的调用封装成了一个统一的接口，例如通过 API 调用时，所有大模型共用同一套调用代码。
