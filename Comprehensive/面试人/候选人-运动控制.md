机器人动力学方程是什么？你能解释每一项的物理意义吗？

动力学方程：  
$$ M(q)\ddot{q} + C(q,\dot{q})\dot{q} + g(q) = \tau $$

- **M (q)**：惯性矩阵（机器人“难动程度”）
- **C (q, dq)**：科氏力、离心力
- **g (q)**：重力
- **τ**：关节输入力矩
- 描述了：**给多大的力矩 → 产生什么样的加速度**
    

⭐ 评价指标
- 是否能清楚解释每一项
- 是否理解“动力学是 FK 的二阶版本”（加分）


**正动力学**：已知机器人的关节驱动力矩和上一时刻的运动状态（角度和角速度），计算得到机器人下一时刻的运动加速度，再积分得到速度和角度；
**逆动力学**：已知某一时刻机器人各关节的位置、关节速度以及关节加速度，求此时施加在机器人各杆件上的驱动力（力矩）

> 逆动力学可以利用牛顿欧拉 (Newton-Euler)方程来求解，也可以利用拉格朗日 (Lagrange)方程来求解

讲一下什么是 CPG，从哪个论文看到的
 Ruppert, F., Badri-Spröwitz, A. Learning plastic matching of robot dynamics in closed-loop central pattern generators. Nat Mach Intell 4, 652–660 (2022).

CPG（Central Pattern Generator）？

**CPG = 中枢模式发生器**，一种**受生物启发的节律信号生成模型**，用于产生

Ruppert, F., Badri-Spröwitz 来自德国斯图加特的马克斯普朗克智能系统研究所，为了去了解生物运动控制器是如何学会利用其腿部力学体现的智能，设计了一个约半米高具有被动弹性的机器人 Morti，在机器人中建立一个 CPG 模型，去学习一个步态策略。该 CPG 模型主要利用脚上接触传感器的稀疏反馈，输出各个关节的运动轨迹，在行走过程中，足底传感数据不断与它自身 CPG 预测的预期触地进行比较，运行反射回路和调整运动轨迹来学习一个步态策略。


通过改进的 Hopf 非线形振荡器构建 CPG 模型，重点关注了被动机械结构（自然动力学）和 CPG 控制器的协同作用，并按时间范围划分反馈活动，在短期内通过短期弹性反馈（反射回路）实现了扰动缓解，这种短期弹性反馈是预先定义的，只有在特定扰动下才会触发，类似于动物的反射功能；与此同时量化了 CPG 控制器和自身机械结构的不匹配，通过贝叶斯优化器可塑性的将 CPG 控制器与自身机械结构相匹配，利用成本函数优化了 CPG 的参数，使 CPG 生成的运动轨迹更适合 Morti 机器人。

 CRBA（Composite Rigid Body Algorithm）是什么？它通常用于什么？

 ✅ 答案要点

- 用于计算惯性矩阵 M(q)
- “合并刚体”算法，像套娃
- 用于：
    - 正动力学（FD）
    - MPC
    - WBC
⭐ 评价指标

- 是否理解 CRBA 在控制中出现的位置
- 是否能区分“逆动力学”和“正动力学”

CRBA 是一种计算机器人惯性矩阵 M(q) 的线性时间算法，通过将每个连杆与其子树的惯性递推合并，形成“复合刚体”，因此叫 Composite Rigid Body。它通常用于正动力学（FD）、MPC 的动态预测、全身控制（WBC）的动力学约束和仿真引擎中，是现代机器人动力学的核心算法之一。

介绍一下分层运动控制模型
VMC 实现身体姿态控制？

**VMC 通过构建虚拟的弹簧、阻尼、扭矩等物理模型，使机器人机体像受到真实力一样运动。**

在姿态控制中，VMC 在身体姿态误差上放置虚拟弹簧和阻尼，然后通过 Jacobian 映射成关节力矩，实现机体回正和稳定。

在与环境交互中，VMC 用虚拟力模拟支撑力、碰撞力、摩擦力，使机器人能够自然地与地面、障碍物或外力互动，保持平衡或柔顺接触。**

## 运动重定向

 Q：假设我们给你一段来自 SMPL 动作序列，你如何将其重定向到一个 28DoF 人形机器人上？

请说出你的 **完整 pipeline**。

**优秀答案结构：**

1. 关节对齐（Human → Robot mapping）
2. 骨长归一化 / 约束归一化
3. IK 或 优化层（如 damped least squares / SQP）
4. 动力学一致性修正（接触点 / CoM / ZMP）
5. 轨迹平滑（velocity/acc smoothing）
6. 输入到 WBC/MPC 执行


Q 2:动物 → 机器人的异构动作映射中，最难的部分是什么？你如何解决？

- 关节拓扑不一致、骨长差异、自遮挡
- 用 **优化式 IK / 时序平滑 / Constraint-based mapping**
- 用 **物理层（WBC/MPC）进行二次修正**
