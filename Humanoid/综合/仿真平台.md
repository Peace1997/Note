## XML


### 属性描述


`contype`：定义几何体的碰撞类型。每个几何体可以有一个或多个碰撞类型，使用位掩码表示。contype 的默认值是 1，表示几何体可以参与碰撞检测。如果设置为 0，则几何体不会参与碰撞检测。

`conaffinity`：定义几何体的碰撞亲和性。每个几何体可以有一个或多个碰撞亲和性，使用位掩码表示。conaffinity 的默认值是 1，表示几何体可以与具有相同碰撞类型的几何体发生碰撞。如果设置为 0，则几何体不会与任何几何体发生碰撞。


## URDF

计算 URDF 整机质量：
```python
import xml.etree.ElementTree as ET 
def compute_urdf_mass(file_path): 
	tree = ET.parse(file_path) 
	root = tree.getroot() 
	total_mass = 0.0 
	for link in root.findall('link'): 
		inertial = link.find('inertial') 
		if inertial is not None: 
			mass = inertial.find('mass') 
			if mass is not None: 
				total_mass += float(mass.get('value')) 
	return total_mass 
if __name__ == "__main__": 
	urdf_file_path = 'robot.urdf' 
	total_mass = compute_urdf_mass(urdf_file_path) 
	print(f'Total mass of the robot is {total_mass} kg')
```
